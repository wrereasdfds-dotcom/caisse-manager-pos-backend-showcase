## Preuve production

J’ai préparé un projet backend structuré comme une base proche production pour un SaaS POS multi-store.

Le projet démontre ma capacité à :
- concevoir une architecture multi-tenant SaaS ;
- développer des APIs REST performantes en Node.js/Express ;
- gérer des flux temps réel POS/KDS/back-office avec Socket.IO ;
- modéliser des données SQL avec PostgreSQL/Prisma ;
- utiliser MongoDB pour les logs/audit events ;
- mettre en place Redis/Bull pour les jobs asynchrones ;
- gérer JWT, validation, sécurité, Docker et CI GitHub Actions ;
- documenter les contrats API et les choix d’architecture.

Le projet n’est pas une application officielle en production, mais un showcase technique complet, aligné avec les problématiques Caisse Manager : caisse, commandes, KDS, stock, paiements, intégrations livraison, multi-store et scalabilité.
