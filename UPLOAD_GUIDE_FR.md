# Guide upload GitHub

1. Dézippe le fichier.
2. Ouvre le dossier extrait.
3. Sur GitHub, crée un nouveau repo :

```text
caisse-manager-pos-backend-showcase
```

4. Ne glisse pas le dossier complet.
5. Ouvre le dossier, fais Ctrl + A, puis glisse directement les fichiers :

```text
README.md
package.json
src/
prisma/
docs/
postman/
.github/
docker-compose.yml
Dockerfile
.env.example
FORM_ANSWERS_FR.md
PROJECT_SUMMARY_FR.md
```

6. Commit message :

```text
Initial POS SaaS backend showcase
```

La page GitHub doit montrer directement README.md, package.json, src, prisma, docs, .github.
